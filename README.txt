About

This module track logs of specific events performed on Node/User/User authentication/Menu/Taxonomy. 
The events are saved in the database and can be viewed on the page admin/reports/track-events.
You could use this to track number of times the CUD operation performed by which users.

Currently, the following events track sub modules are supported:

Node operations (CUD)
User operations (CUD)
User authentication (login/logout/request password)
Menu operations (custom menu's and menu items CUD operations)
Taxonomy operations (vocabulary and term CUD operations)

The event log track can be easily extended with custom events.
